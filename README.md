# Running App

Running App that allows you to track your run path using phone location, save that run and see statistics about all runs


## Technologies used

- MVVM
- Location Tracking
- Dependency Injection - Dagger 2
- Foreground Service
- Room
- Retrofit, OkHttp
- Coroutines
- LiveData
- Permissions
- Navigation Component
- Single Activity


## Api used

I used api from Google Maps API
